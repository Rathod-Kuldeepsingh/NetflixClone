## ✨ Key Features

- 🎯 Exact clone of Netflix's homepage layout
- 🎨 Fully responsive design (mobile + desktop)
- 🎥 Dark-themed UI with hero banner and signup section
- 🔤 Clean HTML semantics & optimized CSS animations
- 🚫 No frameworks or libraries used
